# ✨ 星语 · AI伴侣

沉浸式 AI 虚拟伴侣单页应用。当前版本是一次“大改”：修复设置/模型选择交互，重做模型能力说明、订阅额度 gating、官方伴侣命名与内置 SVG 动漫风立绘。

## 功能亮点

- **设置交互修复**：右上角设置可打开模型与服务商弹窗，服务商卡片、模型卡片、保存按钮和关闭按钮均通过事件绑定工作。
- **模型能力说明**：每个模型展示「擅长什么」「聊天风格」「体验版/Pro 解锁」信息，选择模型时会同步模型说明。
- **订阅额度聊天**：体验版、Pro、∞ 按每日额度限制消息；Pro 模型会在体验版下锁定并引导升级。
- **官方伴侣改名**：小溪→星澜、林风→森屿、梦瑶→晴梨、亦尘→砚澄、清荷→荷音、浩宇→曜野、灵犀→灵绮、沐辰→海澈。
- **8 位手绘伴侣立绘**：不再使用 emoji 作为伴侣头像，改为内置 SVG 动漫风头像/立绘，并为每位伴侣配置不同发型、色彩和专属标记。
- **伴侣推荐模型**：每位官方伴侣绑定推荐服务商与模型，例如星澜使用 DeepSeek Chat，砚澄使用 GLM-4 Plus，灵绮使用 Gemini Flash。
- **品牌化 AI 服务商**：DeepSeek、小米 MiMo、OpenRouter、通义千问、智谱 AI 和自定义 OpenAI 兼容接口均使用独立 Logo Badge 展示。
- **前端 Demo 模式**：填写 API Key 后调用 OpenAI 兼容接口；未填写时自动使用本地 Demo 回复。
- **本地隐私**：年龄验证仅在本地计算；API Key、账号、订阅、心情和聊天记录均保存在 localStorage。

## 使用方式

这是纯前端应用，无需后端服务。

1. 直接打开 `index.html`，或用任意静态服务器托管。
2. 填写出生日期完成本地年龄验证。
3. 选择官方伴侣开始聊天；未配置 API Key 时会进入 Demo 模式。
4. 如需真实模型回复，点击右上角设置，选择服务商 Logo 卡片和模型卡片，并填写 API Key。
5. 伴侣列表、聊天头部和消息历史中的头像/立绘均由页面内置 SVG 绘制，无需外部图片资源。

示例本地服务：

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4173`。

## 数据说明

应用使用以下 localStorage 键：

- `xy2`：主应用状态、会话、订阅、收藏和心情。
- `xyUsers`：演示账号密码，仅用于本地交互原型。

如需重置，可在「我的」页面点击清空本地数据，或手动清理浏览器 localStorage。
