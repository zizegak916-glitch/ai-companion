# ✨ 星语 · AI伴侣

沉浸式虚拟伴侣单页应用。当前版本继续“大改”：去掉外部接口与密钥输入，改为纯前端内置回复引擎；新增豆包模型系列、自定义伴侣、付费墙、订阅额度聊天，并修复 SVG 头像/立绘渲染细节。

## 功能亮点

- **不要外部接口**：页面不再提供密钥输入，也不会请求第三方服务；聊天回复由浏览器内置逻辑生成。
- **内置模型选择**：DeepSeek、豆包、小米 MiMo、OpenRouter 风格、通义千问、智谱均作为“聊天风格/能力”引擎展示。
- **新增豆包**：豆包 Lite 可用于体验版轻聊天，豆包 Pro 用于更强陪伴感、创意续写和日程督促。
- **付费墙**：体验版只开放前 3 位官方伴侣和基础模型；Pro/∞ 解锁高级模型、全部官方伴侣和自定义伴侣。
- **自定义伴侣**：Pro/∞ 用户可在页面中创建自己的伴侣，设置名字、性格、标签、简介和推荐模型；数据只保存在 localStorage。
- **订阅额度聊天**：体验版、Pro、∞ 按每日额度限制消息；额度耗尽或访问锁定角色/模型时会跳到订阅页。
- **官方伴侣改名**：小溪→星澜、林风→森屿、梦瑶→晴梨、亦尘→砚澄、清荷→荷音、浩宇→曜野、灵犀→灵绮、沐辰→海澈。
- **动漫风 SVG 立绘**：伴侣头像和立绘均由页面内置 SVG 绘制，补充 SVG 命名空间并修正立绘发型缩放，不依赖外部图片资源。

## 使用方式

这是纯前端应用，无需后端服务。

1. 直接打开 `index.html`，或用任意静态服务器托管。
2. 填写出生日期完成本地年龄验证。
3. 选择官方伴侣开始聊天；无需配置密钥。
4. 点击右上角设置可切换内置模型系列和模型风格。
5. 进入「订阅」切换 Pro/∞ 后，可解锁高级模型、全部伴侣和自定义伴侣入口。

示例本地服务：

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4173`。

## 数据说明

应用使用以下 localStorage 键：

- `xy2`：主应用状态、会话、订阅、收藏、心情和自定义伴侣。
- `xyUsers`：演示账号密码，仅用于本地交互原型。

如需重置，可在「我的」页面点击清空本地数据，或手动清理浏览器 localStorage。
