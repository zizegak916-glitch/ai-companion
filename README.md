# AI 虚拟伴侣 💕

智能AI虚拟伴侣对话平台。选择一位专属AI伴侣，开始温暖有趣的对话。

## ✨ 功能

- 🔒 **年龄验证** — 依据国家法规，仅限18+用户使用
- 🎭 **6位伴侣** — 温柔、幽默、活泼、稳重、文艺、阳光
- 💬 **自然对话** — 接入 DeepSeek / 小米 MiMo / OpenRouter 等 AI 服务
- 🛡️ **内容安全** — 内置安全规则，杜绝不良内容
- 📱 **移动端适配** — 完美适配手机浏览器
- 🔐 **隐私保护** — 所有数据仅存储在本地浏览器

## 🚀 使用

直接打开 `index.html` 即可使用，无需后端服务器。

1. 点击「开始使用」
2. 填写出生日期完成年龄验证
3. 选择一位AI伴侣
4. 在设置中配置 API Key
5. 开始对话！

## 🔧 支持的 AI 服务

| 服务商 | 模型 |
|--------|------|
| DeepSeek | deepseek-chat, deepseek-reasoner |
| 小米 MiMo | mimo-v2.5-pro |
| OpenRouter | 多种免费模型 |
| 自定义 | 任何 OpenAI 兼容 API |
