# ✨ 星语 · AI伴侣

沉浸式虚拟伴侣单页应用。当前版本继续“大改”：去掉外部接口与密钥输入，改为纯前端内置回复引擎；新增更多内置 AI（Kimi、腾讯混元、MiniMax、阶跃星辰、Gemini、文心一言等）、12 位官方伴侣、自定义伴侣、付费墙、订阅额度聊天，并把头像/立绘强化为偏国漫二次元的全身 SVG 立绘。

## 功能亮点

- **不要外部接口**：页面不再提供密钥输入，也不会请求第三方服务；聊天回复由浏览器内置逻辑生成。
- **内置模型选择**：DeepSeek、豆包、Kimi、腾讯混元、MiniMax、阶跃星辰、Gemini、文心一言、小米 MiMo、OpenRouter 风格、通义千问、智谱均作为“聊天风格/能力”引擎展示。
- **新增更多 AI**：除豆包外，新增 Kimi、腾讯混元、MiniMax、阶跃星辰、Gemini、文心一言等内置风格，并在选择页提供 AI 引擎胶囊筛选。
- **付费墙**：体验版只开放前 3 位官方伴侣和基础模型；点击锁定模型/伴侣或自定义入口会弹出付费墙，而不是直接切换权益。
- **自定义伴侣**：Pro/∞ 用户可在页面中创建自己的伴侣，设置名字、性格、标签、简介和推荐模型；数据只保存在 localStorage。
- **订阅额度聊天**：体验版、Pro、∞ 按每日额度限制消息；额度耗尽或访问锁定角色/模型时会进入订阅页并弹出付费墙。
- **官方伴侣扩容**：保留小溪→星澜等 8 位改名伴侣，并新增墨岚、夏栀、白夜、云舒 4 位官方伴侣。
- **国漫二次元 SVG 立绘**：伴侣头像和卡片立绘均由页面内置 SVG 绘制，强化类似国漫/一人之下气质的墨线描边、大眼高光、全身服装、发型层次、配饰标记和落影，不依赖外部图片资源。

## 付费墙逻辑

- 付费墙是纯前端演示，不接入真实支付；点击“模拟支付并开通”后，会把方案、到期时间和演示收据写入 `localStorage`。
- 体验版：每日 20 条消息、前 3 位官方伴侣、基础模型。
- Pro：每日 200 条消息、全部 12 位官方伴侣、高级模型、自定义伴侣。
- ∞：无限消息、全部 Pro 能力与最高优先级展示。
- 订阅页不再直接切换到付费方案；Pro/∞ 必须经过付费墙确认。

## 使用方式

这是纯前端应用，无需后端服务。

1. 直接打开 `index.html`，或用任意静态服务器托管。
2. 填写出生日期完成本地年龄验证。
3. 选择官方伴侣开始聊天；无需配置密钥。
4. 点击右上角设置可切换内置模型系列和模型风格；在伴侣选择页可用 AI 引擎胶囊快速筛选伴侣。
5. 进入「订阅」或点击锁定内容，会出现付费墙；确认演示支付后解锁高级模型、全部伴侣和自定义伴侣入口。

示例本地服务：

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4173`。

## 数据说明

应用使用以下 localStorage 键：

- `xy2`：主应用状态、会话、订阅、权益到期时间、演示收据、收藏、心情和自定义伴侣。
- `xyUsers`：演示账号密码，仅用于本地交互原型。

如需重置，可在「我的」页面点击清空本地数据，或手动清理浏览器 localStorage。
