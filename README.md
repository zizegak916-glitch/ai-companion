# ✨ 星语 · AI伴侣

沉浸式 AI 虚拟伴侣单页应用。当前版本重点重做了 UI：从深色 Demo 改为更接近正式产品的浅色玻璃拟态界面，并将 AI 服务商入口改成品牌化 Logo 卡片，而不是通用 AI 图标。

## 功能亮点

- **品牌化 AI 服务商**：DeepSeek、小米 MiMo、OpenRouter、通义千问、智谱 AI 和自定义 OpenAI 兼容接口均使用独立 Logo Badge 展示。
- **全新 UI**：首页、伴侣卡片、聊天区、模型设置弹窗、订阅页和个人中心统一为浅色玻璃拟态视觉。
- **8 位手绘伴侣头像**：不再使用 emoji 作为伴侣头像，改为内置 SVG 插画头像，并为每位伴侣配置不同发型、色彩和专属标记。
- **8 位伴侣角色**：温柔倾听、理工幽默、元气陪伴、成熟建议、文艺浪漫、运动打卡、创意脑洞、治愈包容。
- **伴侣探索**：支持关键词搜索、标签筛选、快捷标签和收藏标记。
- **双模式对话**：填写 API Key 后调用 OpenAI 兼容接口；未填写时自动使用本地 Demo 回复。
- **心情签到**：记录今日心情和备注，Demo 回复会结合当天心情生成陪伴话术。
- **聊天管理**：按伴侣保存本地会话、查看消息历史、导出单个会话或全部本地数据。
- **合规与隐私**：年龄验证仅在本地计算；API Key、账号、订阅和聊天记录均保存在 localStorage。

## 使用方式

这是纯前端应用，无需后端服务。

1. 直接打开 `index.html`，或用任意静态服务器托管。
2. 填写出生日期完成本地年龄验证。
3. 选择伴侣开始聊天；未配置 API Key 时会进入 Demo 模式。
4. 如需真实模型回复，点击右上角设置，选择服务商 Logo 卡片并填写 API Key。
5. 伴侣列表、聊天头部和消息历史中的头像均由页面内置 SVG 绘制，无需外部图片资源。

示例本地服务：

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后访问 `http://127.0.0.1:4173`。

## 数据说明

应用使用以下 localStorage 键：

- `xy2`：主应用状态、会话、订阅、收藏和心情。
- `xyUsers`：演示账号密码，仅用于本地交互原型。

如需重置，可在「我的」页面点击清空本地数据，或手动清理浏览器 localStorage。
