## 🚀 使用

纯前端应用，打开 `index.html` 即可使用，无需后端服务器。

1. 打开页面 → 点击「开始体验」
2. 填写出生日期完成年龄验证
3. （可选）注册/登录账号
4. 选择AI伴侣 → 配置 API Key → 开始对话！
